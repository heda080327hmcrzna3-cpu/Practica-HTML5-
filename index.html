Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web moderno.

Comenzar

¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado HTML (HyperText Markup Language), que es el estándar utilizado para crear y estructurar el contenido de las páginas web. Fue desarrollado por la W3C y el WHATWG, y se convirtió en estándar oficial en 2014.

HTML5 no es un lenguaje de programación, sino un lenguaje de marcado. Esto significa que usamos etiquetas para indicarle al navegador cómo mostrar el contenido, como textos, imágenes, videos, formularios y más.

Entre sus principales ventajas están: mejor soporte para multimedia sin necesidad de plugins externos, nuevas etiquetas semánticas que mejoran la organización del código, compatibilidad con dispositivos móviles y mejor integración con CSS3 y JavaScript.

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
  <title>Mi página</title>
</head>
<body>
</body>
</html>

Todo documento HTML5 comienza con la declaración <!DOCTYPE html>, que le indica al navegador que el documento usa la versión 5 de HTML.

La etiqueta <html> es la raíz del documento y contiene todo el contenido. Dentro de ella hay dos partes principales: el <head> y el <body>.

El <head> contiene información que el navegador necesita pero que el usuario no ve directamente, como el título de la página, los estilos CSS y los metadatos. El <body> contiene todo el contenido visible de la página: textos, imágenes, botones, formularios, etc.

Etiquetas principales

h1 - h6

Son etiquetas de títulos o encabezados. h1 es el más importante y grande, y h6 el más pequeño. Se usan para organizar jerárquicamente el contenido.

p

La etiqueta de párrafo se usa para escribir bloques de texto. El navegador agrega un espacio automático antes y después de cada párrafo.

img

Permite insertar imágenes en la página. Usa el atributo src para indicar la ruta de la imagen y alt para describir la imagen en caso de que no cargue.

a

Es la etiqueta de enlace o ancla. Permite crear hipervínculos hacia otras páginas, secciones del mismo documento o archivos. Usa el atributo href para la dirección.

button

Crea un botón interactivo en la página. Puede ejecutar acciones con JavaScript o enviar formularios cuando el usuario hace clic en él.

ul / ol / li

ul crea listas sin orden (con puntos), ol crea listas ordenadas (con números), y li define cada elemento de la lista. Se usan para mostrar información en forma de lista.

HTML5 semántico

Las etiquetas semánticas de HTML5 describen claramente el significado de su contenido, tanto para el navegador como para el desarrollador. Ayudan a organizar mejor el código y mejoran el posicionamiento en buscadores (SEO).

header

Define el encabezado de la página o de una sección. Generalmente contiene el logo, el título del sitio y el menú de navegación principal.

nav

Representa un bloque de enlaces de navegación. Se usa para el menú principal del sitio, permitiendo al usuario moverse entre secciones o páginas.

main

Indica el contenido principal del documento. Solo debe haber un elemento main por página y contiene el contenido más relevante para el usuario.

section

Define una sección temática del contenido. Se usa para agrupar contenido relacionado dentro de la página, como capítulos o partes de un artículo.

article

Representa contenido independiente que podría distribuirse por separado, como una entrada de blog, una noticia o una tarjeta de información.

footer

Define el pie de página del sitio o de una sección. Suele contener información de contacto, derechos de autor o enlaces adicionales.

Formularios en HTML5

Los formularios en HTML5 permiten recopilar información del usuario. Se crean con la etiqueta <form> y dentro de ella se colocan diferentes elementos como campos de texto, botones y áreas de comentarios.

Los elementos más usados en formularios son: input para campos de entrada de datos, label para etiquetar cada campo, textarea para texto largo, select para listas desplegables y button para enviar o ejecutar acciones.

Multimedia en HTML5

Una de las grandes mejoras de HTML5 fue el soporte nativo para multimedia. Antes era necesario instalar plugins como Flash para reproducir videos o audio. Ahora HTML5 permite hacerlo directamente con sus propias etiquetas, sin necesidad de software adicional.

Imágenes

La etiqueta img permite mostrar imágenes en la página. Sus atributos principales son src (ruta de la imagen), alt (texto alternativo) y width/height para el tamaño.

Audio

La etiqueta audio permite insertar archivos de sonido. Con el atributo controls se muestra un reproductor. Soporta formatos como mp3 y ogg.

Video

La etiqueta video permite insertar videos directamente en la página sin plugins. Se pueden agregar controles de reproducción, pausa y volumen con el atributo controls.

Accesibilidad

El atributo alt en las imágenes es fundamental para la accesibilidad. Permite que lectores de pantalla describan la imagen a personas con discapacidad visual.

Conclusión

HTML5 es la base del desarrollo web moderno. Gracias a sus etiquetas semánticas, soporte multimedia nativo y mejor organización del código, se ha convertido en el estándar que todo desarrollador web debe conocer.

A lo largo de esta práctica aprendí que HTML5 no solo sirve para mostrar texto en una página, sino que permite estructurar contenido de forma lógica y significativa, facilitando el trabajo tanto para los navegadores como para los programadores. También comprendí la importancia de separar el contenido (HTML), el diseño (CSS) y la funcionalidad (JavaScript) en archivos diferentes.
